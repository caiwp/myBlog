## 安装

### 安装基本工具

```sh
git clone https://github.com/brendangregg/FlameGraph.git
cp flamegraph.pl /usr/local/bin/
flamegraph.pl -h

go get -v -u github.com/uber/go-torch
go-torch -h

brew install graphviz
```
